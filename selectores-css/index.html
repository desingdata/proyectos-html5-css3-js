Selectores en CSS

Uno de los puntos clave de CSS es saber buscar elementos en el HTML. Esto, se suele hacer mediante CSS, utilizando selectores, pseudoclases y pseudoelementos, o incluso, utilizando nesting.

6 · Pseudoelementos CSS

Precedidos de ::, permiten dar estilo a partes virtuales de un elemento sin alterar el HTML.

Contenido generado: ::before y ::after

Tipográficos: ::first-letter y ::first-line

7 · CSS Nesting Nativo

Permite anidar selectores CSS dentro de otros, como en Sass/Less, pero de forma nativa. Disponible en navegadores modernos desde 2023.

Tarjeta con nesting

8 · Metodología CSS — BEM

BEM (Block · Element · Modifier) es una convención de nombres que evita conflictos de especificidad y hace el CSS predecible y escalable.

Bloque: .tarjeta

Elemento: .tarjeta__texto
